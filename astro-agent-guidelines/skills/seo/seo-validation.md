# Skill: SEO Validation

Ensure pages include essential SEO metadata.

## Tasks

- [ ] Verify title tag on every page
- [ ] Verify meta description on every page
- [ ] Verify heading hierarchy
- [ ] Verify canonical tags if required
- [ ] Verify image alt text

## Verify

- [ ] Metadata visible in page source
- [ ] No missing title or description
- [ ] No skipped heading levels
